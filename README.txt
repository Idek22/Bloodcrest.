Yuka Story Site

WHAT THIS DOES
- Song 1 plays while the reader is inside the Prologue section.
- Song 1 fades out after the prologue break: . . . . . . .
- Song 2 starts when the reader reaches: “YUKA, PLEASE SLOW DOWN,”
- Song 2 fades out after: “Took him long enough to give u-”
- Browsers block automatic audio until the reader presses Start Reading, so the site includes a start button.

HOW TO PUT THIS ON GOOGLE SITES USING OPTION A
1. Upload this folder to GitHub Pages, Netlify, or another static website host.
2. Copy the public website link.
3. Open Google Sites.
4. Click Insert.
5. Click Embed.
6. Paste the public website link.
7. Resize the embed box so the story has enough vertical space.
8. Publish the Google Site.

FASTEST HOSTING OPTION: NETLIFY
1. Go to Netlify.
2. Drag the yuka_story_site folder onto the deploy page.
3. Netlify gives you a website link.
4. Paste that link into Google Sites using Insert > Embed.

FILES
- index.html = story text and page structure
- style.css = visual design
- script.js = scroll-based music behavior
- audio/prologue.mp3 = first song
- audio/chapter1.mp3 = second song
